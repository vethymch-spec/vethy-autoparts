# 欧美 / 日系 工程机械配件图册 — Batch 01

> **使用说明**
> - 所有 OE 号均标注来源层级；**下单前建议按整机 Serial Number (S/N) 再核一次**，CAT / Komatsu / Volvo 同机型不同 S/N 区段零件号常有差异。
> - 来源层级：
>   - **A** — OEM 官方配件手册 / Parts Book (最可靠)
>   - **B** — Donaldson / Baldwin / Fleetguard / WIX / Mann-Filter / HIFI 官方交叉参照
>   - **C** — 行业件号数据库 / 经销商公开清单
>   - **D** — 待核 (结构性占位，不建议直接报价)
> - 配件图片字段留 `image_ref`，后续接图片库时填 CDN 路径。
> - 规模目标：覆盖 10 大品牌 × 7 大类；本批 ≈ 150 行，集中在置信度最高的滤清器 + 典型易损件。
